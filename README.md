MachineLearningAlgorithm
========================
The repro provides the demo program for implementation of machine learning algorithm.

It includes the following algorithms:
  1) kNN
  2) K-Means
  3) Logistic Regression
  4) AdaBoost
  5) SVM
  6) Tree Regression
  7) Decision Tree
  8) Bayis
  9) Apriori
